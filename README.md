# btc-eth-unit-converter

Tool to provide safe conversion between units in BTC and ETH

## Conversions

- weisToGwei
- weisToEth
- gweisToWeis
- gweisToEth
- ethToWeis
- ethToGweis
- satoshisToBtc
- btcToSatoshis
- weisToSatoshis 
- weisToBtc
- ethToSatoshis
- satoshisToWeis
- satoshisToEth
- btcToWeis

For any invalid value (below minimum, negative or base unit decimals) an error will be thrown.
